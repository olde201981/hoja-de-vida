Titulo de CCNA

CCNA Exploration

Routing Protocols and Concepts

   

Description

Describe the purpose, nature and operation of a router and routing tables.

Describe, configure and verify router interfaces.

Explain the purpose and procedure for configuring static routes.

Identify the characteristics of distance vector routing protocols.

Describe the network discovery process of distance vector routing protocols using routing information protocol (RIP).

Describe the functions , characteristics and operation of the RIP portocols.

Compare and contrast classful and classless IP adressing.

Describe classful and classless routing behaviors in routed networks.

Desing and implement a classless IP addressing scheme for a given network.

Demonstrate comprehensive RIP configuration skills.

Describe the main features and operations of the enhanced interior gateway routing protocol (EIGRP).

Describe the basic features and concepts of link-state routing.

Describe the purpose , nature and operation of the Open Shortest Path First(OSPF)protocol.
